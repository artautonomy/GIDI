## How to use GIDI

1. Plug in a MIDI instrument to your PC using a MIDI to USB cable (or for newer MIDI instruments simply a USB-C)
2. Go to [gidi.uk](https://gidi.uk)
3. Select your instrument from **Pad** or **Synth**a
4. Start playing
