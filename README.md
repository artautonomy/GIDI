# GIDI
[GIDI](https://gidi.uk) is a tool for live musicians to augment performances.

## How to use GIDI

1. Plug in a MIDI device
2. Go to [gidi.uk](https://gidi.uk)
3. Select your instrument from **Pad** or **Synth**
4. Start playing
